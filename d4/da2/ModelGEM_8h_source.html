15 // ================================================================================
16 // How are the electron efficiencies obtained?
17 // ================================================================================
18 // Within the scope of two PhD thesis (by Viktor Ratza and Jonathan Ottnad, HISKP University
19 // of Bonn, 2019) models have been derived in order to
20 // describe the collection as well as the extraction efficiencies for GEM
21 // foils. In the following you can find a brief sketch about the procedure behind this.
22 //
23 // Simulations / Measurements:
24 // For different GEM geometries (standard, medium, large pitch) the electric potentials
25 // and fieldmaps were obtained by numerical calculations in Ansys. The fieldmaps were
26 // then used in Garfield++ in order to simulate the drift of the charge carriers
27 // and the amplification processes. In order to obtain the efficiencies, a fixed
28 // amount of electrons has been randomely distributed above the GEM for the simulations.
29 // The efficiencies were thereupon derived by counting where the initial electrons and
30 // electrons from the amplification region ended, e.g. Copper top / bottom, anode etc.
31 // Indeed the simulated efficiencies are in a good agreement to measurements.
32 //
33 // Calculations [2]:
34 // In oder to get an analytic understanding of the efficiencies, a simplified and
35 // two-dimensional model has been investigated. Simplified means: No differentian
36 // between an inner and an outer diameter, no Polyimide layer, no gas/no diffusion
37 // and two-dimensional cut of the hexagonal 3D GEM structure.
38 // The resulting equations are in a good agreement to the results from the simulations
39 // in terms of limits, offsets and curves for different pitches (in case of no diffusion).
40 // Nevertheless differences can be found due to the simplifications in the model.
41 // By introducing three fit parameter, the calculations can be tuned
42 // in a way to describe the simulated datapoints. The resulting equations
43 // describe the efficiencies in a full region and for different GEM pitches.
44 //
45 // The results from the fitted equations are implemented in this class for NeCO2N2 90-10-5.
